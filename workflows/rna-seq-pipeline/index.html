RNA-seq pipeline

Bulk RNA-seq from FASTQ to differential expression using STAR, featureCounts, and DESeq2. Supports single-end and paired-end reads, multiple reference genomes, and multi-group comparisons.

Quick start

# 1. Clone the workflow
git clone https://github.com/unsw-restech/ares.git
cd ares/workflows/rna-seq-pipeline

# 2. Edit parameters
cp params.example.yml params.yml
nano params.yml

# 3. Run on Katana
nextflow run main.nf \
  -profile katana \
  -params-file params.yml

Parameters

Parameter Type Required Default Description
reads path required — Path to FASTQ files (glob pattern)
genome string required GRCh38 Reference genome (GRCh38, GRCm39, mm10)
strandedness string optional auto Library strandedness (auto, forward, reverse, unstranded)
min_mapped_reads integer optional 5000000 Minimum mapped reads threshold per sample
outdir path optional ./results Output directory

Example params.yml

reads: "/srv/scratch/zXXXXXXX/my-project/fastq/*_R{1,2}.fastq.gz"
genome: "GRCh38"
strandedness: "auto"
outdir: "/srv/scratch/zXXXXXXX/my-project/results"
